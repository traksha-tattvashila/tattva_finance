## Config location

`artifacts/tattva-finance/capacitor.config.ts`

- appId: `com.tattvashila.finance`
- appName: `Tattva Finance`
- webDir: `dist/public` (matches Vite build output)

## Android project location

`artifacts/tattva-finance/android/` — committed to repo, generated by `npx cap add android`

## Build flow

1. `PORT=4173 BASE_PATH=/ pnpm --filter @workspace/tattva-finance run build` — builds web assets
2. `cd artifacts/tattva-finance && npx cap sync android` — copies web assets into android project
3. `cd artifacts/tattva-finance/android && ./gradlew assembleDebug` — builds APK

APK output: `artifacts/tattva-finance/android/app/build/outputs/apk/debug/app-debug.apk`

## GitHub Actions

`.github/workflows/build-apk.yml` — triggers on push to main; uses pnpm, Java 17, android-actions/setup-android@v3

## vite.config.ts env vars

PORT and BASE_PATH now have defaults (4173 and /) so build works without Replit env vars. The Replit workflow still sets them via `[services.env]` in artifact.toml.

**Why:** Strict throws broke CI/GitHub Actions builds and Replit publish builds where env vars weren't injected.

## PWA assets

- `public/manifest.json` — PWA manifest (theme: #5B21B6)
- `public/sw.js` — cache-first service worker, registered in main.tsx (prod only)
- `public/icons/icon-192.png`, `icon-512.png`, `icon-maskable-512.png` — purple T-logo icons
